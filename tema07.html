DRINKS DATABASE

Learning to implement a JSON file we chose a Drinks Database as out main frame.
To design the logo and come up with a name we researched existing examples. Afterwards we put on a time-limit and sketched down as many ideas as we could in the given timeframe.

The process also included creating and photographing the products, which included its own research and preparation.
Going through a few user tests early on, we discovered that there were a few off colours in our colour palette, and we therefore redesigned the site with lesser colours in mind.
